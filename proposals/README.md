# プロダクト提案：AI Coordinator（方向性B）

## 元のアイデア
「CrossBar」ver2の「テキスト入力ゼロ」というコンセプトをベースに、さらにユーザーの「探す」という能動的な負荷すらも排除し、AIが最適なマッチングをコーディネートする方向性です。

## 提案一覧

| 提案 | コンセプト | 特徴 | 向いている人 |
|------|------------|------|--------------|
| A | **The Swipe** | マップを廃止し、AIが選んだ候補を1人ずつスワイプで選ぶ | 「誰がいるか探す」ことすら面倒な、究極の受動型ユーザー |
| B | **Smart Spotlight** | マップ上にAIが「今会うべき人」をスポットライト表示する | マップで街の雰囲気を感じつつ、最適な出会いも逃したくない人 |
| C | **Proactive Matchmaker** | AIがマッチングを検知し、双方に「合流しませんか？」と通知 | 社内の交流活性化を最優先し、AIに背中を押してほしい企業 |

## 次のステップ
1. 各提案（proposal-a.md 〜 c.md）を見て、気になるものを選んでください
2. 「Aのシンプルさがいい」「BをベースにCの要素を入れたい」など教えてください
3. 選んだ方向性を深掘りし、最終的な要件定義に進みます
