# プロダクト要件

## 1. プロダクト概要
### プロダクト形態
モバイルアプリ（iOS / Android）

### 利用環境
夜間の移動中、または飲食店での滞在中。

## 2. コアバリュー
### このプロダクトの核となる価値
「今、近くにいる」という事実を、AIの力で「ビジネスの好機（Cross）」に変える。

### Aha Moment
マップを開いた瞬間、自分と相性の良い人がスポットライトで光り、「Cheers!」をタップするだけで会う約束が成立した時。

## 3. 主要機能
### 必須機能（Must Have）
- 位置情報マップ表示（Smart Spotlight対応）
- マジック・トグル（ステータス切り替え）
- ワンタップ・オファー（Cheers / SOS / Ticket）
- 動的な情報開示制御（Visibility Matrix）

### 推奨機能（Should Have）
- 提携店舗のワンタップ予約
- 共通の話題（Contextual Tags）表示

## 4. 非機能要件
### パフォーマンス
- 位置情報のリアルタイム反映（遅延5秒以内）。

### セキュリティ
- 位置情報の暗号化。
- 企業ドメインによる所属認証。
- ボスブロック（特定の相手への非表示設定）。

### スケーラビリティ
- 初期は特定コミュニティ（数千人規模）を想定。
