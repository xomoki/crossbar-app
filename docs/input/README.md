# CrossBar (Smart Spotlight Edition)

## 概要
テキスト入力ゼロで、AIが「今会うべき人」をマップ上にスポットライト表示する、経営者・社員向け位置情報活用型SNS。

## 解決する課題
- **誘う心理的ハードル**: 文章を考えたり、断られる恐怖をAIと直感的UIで排除。
- **組織のサイロ化**: 社内のタテ（役員×社員）とヨコ（他部署間）の偶発的な接点を創出。
- **経営者の孤独**: 会食後の空き時間を、質の高い社外交流や社内メンタリングに変える。

## ターゲットユーザー
- 経営者（会食後の高品質な交流を求める層）
- ハイポテンシャル社員（キャリア相談や他部署交流を求める層）

## 主要機能
- **Magic Toggle**: 言語化不要のステータス切り替え（Open/Listen/Biz/Ghost）。
- **Smart Spotlight**: AIが相性の良い相手をマップ上で光らせてレコメンド。
- **One-Tap Offer**: 「Cheers!」や「Ticket」などのアイコンを飛ばすだけでお誘い完了。
- **Visibility Matrix**: 関係性に応じた位置情報の動的曖昧化（プライバシー保護）。
